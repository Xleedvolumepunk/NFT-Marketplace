NFT Marketplace — это децентрализованная платформа для создания, покупки, продажи и аукционной торговли NFT (невзаимозаменяемых токенов). В основе лежат смарт-контракты на Solidity, а для взаимодействия с контрактами используется Web3 и ethers.js.

Функционал

Создание NFT: Поддержка стандартов ERC-721 и ERC-1155.

Листинг NFT: Пользователи могут выставлять NFT на продажу.

Покупка и продажа: Торговля происходит через смарт-контракты, обеспечивая прозрачность сделок.

Аукционы: Пользователи могут устраивать аукционы на NFT.

Роялти для создателей: Авторы NFT получают процент от повторных продаж.
